Body Parsing

Body parsing, det er vigtigt, fordi når en klient sender data til serveren, som modtages i JSON-format, så skal vi konvertere så vi kan arbejde med dataen. Det er hver gang der sendes request til vores server som en del af en request i Express, så er det nødvendigt at parse dataene fra requesten for at kunne bruge dataen. Det er en måde at parse data fra JSON-format og det der sker er at vi parse data fra JSON-format og konvertere det om til JavaScript objekter. I Express er der indbygget middleware kaldet for express.json(), som parse data fra JSON-format, og konvertere det om til JavaScript objekter. Ved at bruge app.use(express.json()) i vores Express-applikation konfigurerer vi Express til automatisk at parse request body fra JSON og konvertere det om til et JavaScript-objekter. Det betyder, altså, når en klient sender JSON-data til vores server, så vil vi kunne få adgang til dataene som JavaScript-objekter i req.body, inde i vores route funktioner.

Det parse request body, fordi så kan vi manipulerer og arbejde med de indgående data på en strukturet måde. Uden body parsing, så skal vi manuelt udføre parsing, som er besværligt og man opstå fejl under processen.
Ved at bruge Express indbygget body parsing middleware, så kan vi effektivt håndterer indgående data fra klienter og bygge mere robuste og strukturerede server applikationer.
