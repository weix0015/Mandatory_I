JavaScript Datatyper

JavaScript har forskellige datatyper: String, Number, Boolean, Symbol, BigInt, Null, Undefined, Object, Arrays, Function. Der er 7 primitive datatyper i javascript, String, Number, Boolean, Symbol, BigInt, Null, Undefined og 1 ikke primitive datatyper i javascript, som er Object, der er også Array, som er en type Object og det samme gælder for Function, men hvis man bruger typeof og console.log() så får man Function i stedet for Object.

Variabler, String, Numbers og floats

Man opretter variabler med const eller let. const det er konstant, men ikke i dens værdi, men i tildelingen, som betyder, variablen kan ikke blive tildelt igen og at den skal være tildelt til at starte med. Derimod med let kan variablen godt tildeles igen, og man skal undgå at bruge var på grund af det kan skabe side effects og var har function scope, imodsætning til const og let, som har blockscope, hvilket gør man kun har adgang inden i blokken. Her er et eksempel: 
{ 
  let x = 10;
  console.log(x); // Output: 10 
}

console.log(x); // ReferenceError: x is not defined
Her er et eksempel på side effect, hvor man aldrig skal bruge var 
for (var i = 0; i < 5; i++) {
  setTimeout(function() {
    console.log(i); // Output 5 five times
  }, 1000);
}
Her er et eksempel på const, som ikke kan blive tildelt igen til en variable. 
const x = 10;
x = 20; // TypeError

Arrays og for loop

I JavaScript, får man adgang til elementer i et array ved hjælp af at bruge push(), pop(), splice() og man bruger til at fx hvis man vil kopier et array, men der skal kun bruge for loop hvis vi skal tælle noget, eller så skal man bruge loop metoder i stedet for, så man undgår side effects. Her er eksempler på push(), pop(), splice() og for loop 
const array = [1, 2, 3];

array.push(4); // adds 4 to the end of the array
console.log(array); // Output: [1, 2, 3, 4]
        
const poppedElement = array.pop(); // remove the last element (4)
console.log(poppedElement); // Output: 4
console.log(array); // Output: [1, 2, 3]
        
const removedElement = array.splice(1, 1); // remove the element at index 1 (2)
console.log(removedElement); // Output: [2]
console.log(array); // Output: [1, 3]

Objects

Hvis vil have adgang til værdier i et objekt, så skal man gøre det ved hjælp af keys. Man opretter et objekt med key-value-pair, og kan tilføje og fjerne egenskaber fra et objekt. Her er nogle eksempler på at få adgang til et objekt, og oprette et objekt ved hjælp af key-value-pair og tilføje og fjerne egenskaber fra et objekt. 
// Eksempel på, at få adgang til values inde i et objekt ved hjælp af at bruge keys:
const obj = { a: 1, b: 2, c: 3 };

Object.keys(obj).forEach(key => {
  console.log(obj[key]); // Output 1, 2, 3 (values for keys "a", "b", "c")
});

// Eksempel på, at oprette et objekt med key-value-pair
const obj = { key1: "value1", key2: "value2" };
console.log(obj);  // Output: { key1: "value1", key2: "value2" }

// Eksempel på, at tilføje og fjerne egenskaber fra et objekt
const obj = { a: 1 };

// adding an attribute
obj.b = 2;
console.log(obj); // Output: { a: 1, b: 2 }

// removing an attribute
delete obj.a;
console.log(obj);  // Output: { b: 2 }
