JavaScript Function

I javascript er der hoisting, som betyder at funktioner og variabler kan tilgås, før de er deklaret. Altså de bliver løftet op til toppen af deres scope før alle code execution.

Hoisting 

let word;
console.log(word); // Output undefined
word = "Hello";
console.log(word); // Output "Hello"
Her er et eksempel, hvor word bliver først decleared efter output undefined, og så output den igen, hvor resultatet vil være "Hello".

I javascript er der forskellige funktioner.

Anonymous Function 
const getRandomIntAnonymousFunction = function (min, max) {
  return Math.floor(Math.random() * (max + 1 - min) + min);
};
Det er et eksempel på en Anonymous function, som har 2 parameter, det er 2 tal som input, hvor funktionen returnerer et tilfældigt helttal. Den bruger Math.random() som bruges til at generere et tilfældigt decimaltal mellem 0 og 1, og derefter omregnes det til et heltal ind i intervallet. Anonymous funktioner er funktioner uden et navn. De oprettes direkte som en værdi og tildeles ofte til enten variabler eller bruges som argumenter til andre funktioner. funktioner kan udføre en handling, men da de ikke har noget navn, så kan de ikke kaldes udenfor det scope, som de er defineret, medmindre de er tildelt til en variabel.

Arrow Function 
const getRandomIntArrowFunction = (min, max) => {
  return Math.floor(Math.random() * (max + 1 - min) + min);
};
Det er et eksempel på en arrow function. I javascript kan en arrow function på en enkelt linje, altså hvis den kun indeholder en instruktion, kan { } og return keyword udelades, som gør koden mere kompakt. Det kan også skrives i flere linjer med { } og return keyword, men det behøves det ikke hvis det kun indeholder 1 instruktion. I det her eksempel er det en arrow function, der genere et tilfældigt heltal inden for et givet interval min og max. Den bruger Math.random() og Math.floor() metoderne til at genere et random heltal. 
const calculateArea = (width, height) => {
  const area = width * height;
  return area;
};

console.log(calculateArea(5, 3)); // Output 15
Her er et eksempel på arrow function med { } og return keyword, hvor det ikke er en one liner. Det tager imod 2 parametre width og height. Funktionen beregner arealet ud af en rektangel ved multiplikation af bredden med højden og returnerer resultatet.

Callback eller callback-function 
function performAction(callback) {
  callback();
}

function greet() {
  console.log("Hello!");
}

performAction(greet); // Output "Hello!"
Det er et eksempel på callback / callback-function. Det er en reference til executeable code, der sendes som et argument til en anden del af koden. I det her eksempel, så bliver der kaldt på performAction function med greet som input, greet funktionen console.log("Hello!") ud så når performAction bliver kaldt med greet som input, vil funktionen performAction udføre det, hvor der er givet callback som input, så vil den output "Hello!" med andre ord. 
function performAction(callback) {
  callback(); // 3. Execute the function provided as input (callback)
}

function greet() {
  console.log("Hello!"); // 2. Defines the action to output "Hello!"
}

performAction(greet); // 1. Calls performAction with greet as input
