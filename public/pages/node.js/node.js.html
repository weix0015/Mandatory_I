Node.JS

Node.js det er et runtime-enviroment, der gør man kan køre javascript kode i terminalen eller i en form for konsole, uden for browser. Man skal først installerer node.js, som man kan gøre på node.js officielle hjemmeside hvor der er forskellige versioner, hvor der både er support til Windows, MAC og Linux. Derefter kan man køre javascript kode enten ved at skrive javascript kode i en konsole, terminal eller en andre slags command-line-interface. Man kan også kører javascript, hvis man har en javascript fil, så skal man være inde i samme lokation som filen og så skal man skrive node app.js. Man skriver node og navnet på filen.


Du kan download node.js på deres officielle website, ved at trykke på billedet, som fører dig hen til nodejs website
