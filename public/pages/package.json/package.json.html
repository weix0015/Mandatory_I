Package Json

Package Json, det er en fil, der bruges i Node.js-projekter (package.json), som bruges til at holde styr på projektets afhængigheder (dependencies) og konfigurationer.
Der er et værktøj, som hedder npm, som står for Node Package Manager, som følger med node.js og det bruges til at administrere og installere pakker (biblioteker og moduler) til node.js-projekter.
Der skal oprettes en package.json fil ved at køre npm init i terminalen i projektmappen. Det vil starte en interaktiv proces, hvor du kan konfigurere dit projekt og oprette en package.json fil.
Man kan tilføje de pakker, man har brug for, til dependencies sektionen i package.json filen. Efter man har tilføjet alle ens afhængigheder / dependencies, så kan man køre npm install i terminalen i ens projektmappe og npm vil derefter automatisk installere alle de specificerede pakker og oprette en node_modules mappe i projektmappen. Eksempel på package.json

package.json 

{
  "dependencies": {
    "express": "4.18.2"
  }
}
Det er et eksempel på indholdet i en package.json fil. I det her tilfælge defineres afhængighederne for projektet, hvor express modulet er specificerede som en afhængighed, med version "4.18.2". Det betyder altså, når kommandoen npm install bliver tastet i terminalen i ens projektmappe, så vil npm automatisk installere express modulet i projektet, så det kan bruges i koden.
JSON det er en format, en fil type og det står for JavaScript Object Notation. Det er et nemt læseligt dataudvekslingsformat, som bruges til at sende og modtage data. Det er samme format der bruges til at skrive package.json filer og konfigurationsfiler i mange andre programmeringsprog, teknologier og værktøjer.
